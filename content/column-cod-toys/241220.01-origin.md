## 背景

開始前來簡單了解一下相關背景。

- PowerToys

  [PowerToys](https://learn.microsoft.com/zh-tw/windows/powertoys/) 是微軟釋出的一個開源專案，提供了一系列的 Windows 工具。

  有很多實用的功能，例如：[顏色選擇器](https://learn.microsoft.com/zh-tw/windows/powertoys/color-picker)、[找不到命令](https://learn.microsoft.com/zh-tw/windows/powertoys/cmd-not-found)等等。

  其中的 [PowerToys Run](https://learn.microsoft.com/zh-tw/windows/powertoys/run) 工具是我們的主要靈感來源，基本上借鑑其功能。

- Electron

  [Electron](https://www.electronjs.org/) 是一個可以用 HTML、CSS、JavaScript 建立跨平台桌面應用程式的框架。

  此框架由 GitHub 開發，有許多知名應用程式也用 Electron 開發，例如：[VS Code](https://code.visualstudio.com/)、[Slack](https://slack.com/)、[Discord](https://discord.com/) 等等。

## 特點

目前主要功能規劃與特點如下：

- 鍵盤優先
  呼叫工具、選擇功能皆使用鍵盤操作，可以在不中斷輸入的情況下快速使用

- 根據關鍵字推薦功能
  根據輸入的關鍵字候選功能

- 功能：顏文字庫
  從 Notion Database 取得顏文字資料

介面則模仿 [PowerToys Run](https://learn.microsoft.com/zh-tw/windows/powertoys/run) 的啟動器，只要按下 Alt + Space 就會跳出如下圖輸入框。

### 在現有 Vue 專案中加入 Electron

雖然官方推薦使用 [Electron Forge](https://www.electronforge.io/) 建立專案，不過我已經有一個 Vue 模板，所以這次使用 `vite-plugin-electron` 來整合 Electron。

步驟沒什麼特別的部分，基本上完全依照[文件](https://github.com/electron-vite/vite-plugin-electron)操作。

一陣猛如虎的操作後，我們得到了[初始環境](https://github.com/Codfisher/side-project-cod-toys/tree/init)惹！<span class="text-nowrap">ˋ( ° ▽、° )</span>

### 快捷鍵開啟視窗

第一步當然是先做一個簡單的輸入框。

`src\pages\index.vue`

```vue
<template>
  <div class="flex-col">
    <q-input
      v-model="inputValue"
      placeholder="要來點甚麼？...(´,,•ω•,,)"
      autofocus
      outlined
      square
    >
      <template #prepend>
        <q-icon name="search" />
      </template>
    </q-input>
  </div>
</template>

<script setup lang="ts">
import { ref } from 'vue'

const inputValue = ref('')
</script>
```
