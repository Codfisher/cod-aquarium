# 臉被切到了怎麼辦！來自定義 markdown 語法吧

![markdown-it-custom-syntax](/markdown-it-custom-syntax.webp)

段落文字太長時，在句尾的顏文字很容易被換行，像這樣 (＃°
Д°)

一秒便變成了一坨不知所云的亂碼。( ´•̥̥̥ ω •̥̥̥` )

我原本的解法是這樣：
::: raw
<span class="text-nowrap">(›´ω`‹ )</span>
:::

把顏文字用獨立區塊包起來，雖然有效，但是不夠優雅。
