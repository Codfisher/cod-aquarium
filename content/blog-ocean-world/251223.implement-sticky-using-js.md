## 甚麼是 sticky

sticky 是一種特殊的定位方式，結合了 relative 和 fixed 的特性。

當元素滾動到指定位置時，會「黏住」並固定在視窗或滾動容器內的某個位置，直到其父容器的邊界被觸發。

<css-sticky />

嘗試滾動上面這個範例，可以發現 sticky 元素會黏在視窗的頂部，不會像 relative 元素一樣滾出視窗。

好用是很好用，但是 CSS sticky 也有限制，像是多層滾動區域無法正常工作：

<css-sticky-no-effect />
